NETWORK TOPOLOGIES

Introduction Network-type Network-topology Advantage & Disadvantege Summary

Network topology refers to the layout in which the various components of a network,
like nodes, links, peripherals, etc., Are connected and communicate with each other. A
topology can either be physical or logical. Physical topology is the physical layout of
nodes, workstations, and cables in the network, whereas logical topology is the way
how information flows between the different components.

Network topologies are categorised into the following basic types:

POINT-TO-POINT

Point-to-point isÂ a simple topology that directly links two nodes
and reserves the entire bandwidth of the connection for them
to communicate with each other. This type of network is more
suitable for small areas where computers are in close
proximity. This technology provides a faster and reliable
connection.

BUS TOPOLOGY

This topology uses a single communication line or one main cable to which all the nodes are directly connected. The main cable acts as a backbone for the network. It transmits data only in one direction.
This type of structure, one of the computers in the network acts as the computer server, that provides data to all the clients. This topology is used in small network where cable requirements is relatively small. For this reason, this type of topology is not used for large networks, such as those covering an entire building.

STAR TOPOLOGY

A Star topology is a type of network topology in which all the devices or nodes are physically connected to a central node such as a router, switch, or hub. This devices communicate across the network by passing data through this hub. The central node (hub) acts as a server, and the connecting nodes act as clients. Star topology is very popular because the setup cost is low. It is easy to add a new device to the network as only one cable is required and the configuration is simple.

RING TOPOLOGY

A ring topology is a type of network configuration where each device is connected directly to two other devices, forming a single continuous pathway for data transmission. In a ring topology, data travels in one direction around the ring until it reaches its destination or encounters a break in the ring. Ring topologies are used in both LAN and WAN setups. The main disadvantage of a ring topology is its susceptibility to network failure. If any single device or connection within the ring fails, it can disrupt the entire network gets affected.

TREE TOPOLOGY

A tree topology in computer networking is a hierarchical arrangement of nodes in which each node is connected to a central node, often referred to as the root node or the hub. The structure resembles a tree, with branches stemming out from the central node to connect to other nodes. Tree topologies are commonly used in wide area networks (WANs) and local area networks (LANs), particularly in situations where scalability and ease of management are important. In tree topology, the expansion of the network is possible and easy but maintenance becomes difficult.

MESH TOPOLOGY

Mesh topology is a network topology in which each node (device or computer) is connected to every other node in the network, forming a fully interconnected mesh of communication links. In a mesh topology:

1. Direct Connections: Each node has a dedicated point-to-point link to every other node in the network. These links can be physical cables or virtual connections.

2. Scalability: Mesh networks can be easily scaled by adding more nodes without disrupting the existing network structure. Each new node simply needs to establish connections with the existing nodes.

Mesh topologies are commonly used in scenarios where reliability and fault tolerance are critical, such as in large-scale networks, military applications, and some types of telecommunications networks.
